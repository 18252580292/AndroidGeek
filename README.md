## 主要技术 ##

- MVP架构
- Retrofit + Okhttp
- Tinker 热更新
- RxJava + RxAndroid
- EventBus
- Glide
- Dagger2
- Material Design
- Logger
- butterknife
- circleimageview
- BaseRecyclerViewAdapterHelper
